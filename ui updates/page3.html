#PYTHON #FUNCTIONS

Objective

Define 'Recursion' in the context of functions.

key

Answer Decrypted

A process in which a function calls itself as a subroutine.

This allows the function to repeat its behavior until a specific condition (base case) is met.

def factorial(n):
  if n == 1: return 1
  return n * factorial(n-1)
